Salle 1 Analyse du réseau 

On commence dans un aéroport avec le matériel suivant : 
	- Appareil photo (mode thermique, infra rouge, vision nocturne)
	- Appareil GPS 
	- Radio 
	- Carte 

Deux photos de ciel, une avec le drone caché, l'autre non, une fois découvert, l'autre doit prendre l'appareil GPS pour taper le code

- Prendre le drone en photo avec la vision thermique 
- Celui qui a pris le drone en photo donne le code à l'autre qui le rentre dans l'appareil GPS 
- Ajouter le code dans l'appareil GPS pour récupérer le signal 
- Analyser le signal pour trouver des coordonnées (latitude/longitude) sur une carte

- L'un va avoir des chiffres dans le désordre et va devoir les dire à l'autre qui va tester les chiffres un à un pour avoir le code final et arrêter l'analyse en cours
- On découvre que l'entreprise cherche le drone qui a été pris en photo
